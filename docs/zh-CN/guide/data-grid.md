# 数据表格

DBX 的数据表格组件用于浏览和编辑表中的数据，支持大数据量虚拟滚动、行内编辑、排序过滤以及多种格式导出。

## 虚拟滚动

数据表格采用虚拟滚动技术，仅渲染当前可视区域内的行，即使表中有数十万条记录也能保持流畅的滚动和操作体验。

## 行内编辑

你可以直接在表格中编辑数据，无需编写 SQL 语句：

- **修改数据** -- 双击单元格即可编辑，修改后的单元格会高亮标记
- **新增行** -- 点击工具栏中的"新增行"按钮，在表格底部插入一行空记录
- **删除行** -- 选中行后点击"删除"按钮，标记该行待删除

所有修改会暂存在本地，不会立即写入数据库。

## SQL 预览

在点击"保存"之前，你可以预览 DBX 即将执行的 SQL 语句（`UPDATE`、`INSERT`、`DELETE`），确认无误后再提交。这样可以避免误操作对数据造成不可逆的影响。

## 排序与过滤

- **排序** -- 点击列头可按该列升序或降序排列
- **搜索** -- 使用搜索框快速查找包含指定文本的行
- **WHERE 过滤** -- 输入自定义的 `WHERE` 条件，精确筛选满足条件的数据

## 列宽调整与行号

- 拖拽列头边缘可以调整列宽，以适应不同长度的数据内容
- 表格左侧显示行号，方便定位和引用

## 数据导出

选择工具栏中的导出功能，可将当前表格数据导出为以下格式：

| 格式 | 说明 |
|---|---|
| CSV | 逗号分隔的文本文件，适合导入 Excel 或其他工具 |
| JSON | JSON 数组格式，适合程序处理 |
| Markdown | Markdown 表格格式，适合粘贴到文档中 |
