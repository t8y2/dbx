# 数据传输

DBX 支持在不同数据库引擎之间传输数据，例如将 MySQL 中的数据迁移到 PostgreSQL，或从 SQLite 导入数据到 SQL Server。

## 使用方式

1. 打开数据传输功能
2. 选择**源连接**和**目标连接**（可以是不同类型的数据库引擎）
3. 选择要传输的表
4. 点击"开始传输"

DBX 会自动处理不同数据库之间的类型映射和语法差异。

## 选择要传输的表

你可以选择传输源数据库中的全部表，也可以手动勾选需要传输的特定表。对于不需要迁移的表，取消勾选即可跳过。

## 进度追踪

传输过程中，DBX 会实时显示：

- 当前正在传输的表名
- 已传输的行数
- 整体传输进度

你可以随时查看传输状态，了解剩余工作量。

## 错误处理

如果传输过程中遇到错误（如类型不兼容、约束冲突等），DBX 会：

- 记录错误信息和对应的表名
- 继续传输其他表，不会因单个表的错误中断整个任务
- 传输完成后汇总展示所有错误，便于排查和处理
