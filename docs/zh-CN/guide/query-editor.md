# 查询编辑器

DBX 内置了基于 CodeMirror 6 的 SQL 查询编辑器，提供语法高亮、智能补全和快捷操作，帮助你高效编写和执行 SQL。

## 语法高亮

编辑器针对不同的 SQL 元素（关键字、表名、字符串、数字、注释等）提供语法着色，使 SQL 语句结构一目了然。高亮规则会根据当前连接的数据库类型自动适配。

## SQL 自动补全

在编辑器中输入时，DBX 会自动弹出补全建议，支持以下内容：

- **关键字** -- `SELECT`、`FROM`、`WHERE`、`GROUP BY` 等 SQL 关键字
- **表名** -- 当前数据库中的所有表和视图
- **字段名** -- 根据已输入的表名，自动补全对应的字段
- **JOIN 补全** -- 输入 `JOIN` 后，自动建议可关联的表及关联条件

补全列表会随着输入内容实时过滤，按 `Tab` 或 `Enter` 即可确认选择。

## 执行查询

DBX 提供多种执行方式：

| 操作 | 快捷键 |
|---|---|
| 执行当前编辑器中的全部 SQL | `Cmd+Enter`（macOS）/ `Ctrl+Enter`（Windows/Linux） |
| 仅执行选中的 SQL 片段 | 先选中文本，再按 `Cmd+Enter` |

执行结果会显示在编辑器下方的结果面板中，包括返回的数据行、执行耗时和影响行数。

## SQL 格式化

点击工具栏中的格式化按钮，可以将当前 SQL 语句自动格式化为统一的缩进和换行风格，便于阅读和维护。

## 查询历史

DBX 会自动记录你执行过的 SQL 语句。你可以：

- 在查询历史面板中浏览之前执行过的所有 SQL
- 点击某条历史记录，将其填入编辑器重新执行
- 按时间顺序查看历史记录
