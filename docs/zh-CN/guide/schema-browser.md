# 结构浏览

DBX 提供树形结构浏览器，让你可以快速浏览和管理数据库中的所有对象。

## 树形视图

侧边栏以树形层级展示数据库结构，层级关系如下：

```
连接
 └── 数据库
      └── Schema
           └── 表 / 视图
                └── 字段
```

点击节点可展开或折叠下级内容。点击表名可直接打开该表的数据浏览视图。

## 搜索与置顶

- **搜索** -- 在侧边栏顶部的搜索框中输入关键词，可快速过滤显示匹配的表、视图或字段
- **置顶** -- 将常用的表或连接置顶显示，方便快速访问

## 右键上下文菜单

在树形视图中右键点击节点，可以调出上下文菜单，根据节点类型提供不同的操作选项，例如：

- 打开表数据
- 查看表结构
- 复制表名
- 刷新节点
- 删除表

## 字段注释显示

如果数据库中的字段设置了注释（COMMENT），树形视图中会在字段名旁边显示对应的注释文本，帮助你理解字段含义。

## 连接颜色标记

你可以为每个数据库连接设置不同的颜色标记。颜色会显示在侧边栏的连接名称旁，便于在多个连接之间快速区分，尤其适用于区分开发环境和生产环境。
