# Schema 对比

DBX 的 Schema 对比功能可以比较两个数据库之间的结构差异，并生成同步 SQL，帮助你快速对齐数据库结构。

## 对比两个数据库

1. 打开 Schema 对比功能
2. 分别选择**源数据库**和**目标数据库**连接
3. 点击"对比"按钮，DBX 将自动分析两个数据库中所有表和字段的差异

对比结果会以清晰的列表展示：

- **仅在源库存在的表** -- 目标库中缺少的表
- **仅在目标库存在的表** -- 源库中没有的表
- **结构不同的表** -- 字段类型、默认值、索引等存在差异的表

## 生成同步 SQL

对比完成后，DBX 会根据差异自动生成同步 SQL 语句，包括：

- `CREATE TABLE` -- 创建目标库中缺少的表
- `ALTER TABLE` -- 修改存在差异的列、索引等
- `DROP TABLE` -- 删除目标库中多余的表（可选）

你可以查看生成的 SQL 内容，确认每条语句的变更是否符合预期。

## 执行同步

确认同步 SQL 无误后，可以直接在 DBX 中执行这些语句，将目标数据库的结构同步为与源数据库一致。无需手动复制 SQL 到其他工具中运行。
