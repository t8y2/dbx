# SQL 文件执行

DBX 支持直接执行 `.sql` 文件，适用于运行数据库迁移脚本、初始化脚本或批量 SQL 操作。

## 执行 SQL 文件

1. 打开 SQL 文件执行功能
2. 选择要执行的 `.sql` 文件
3. 选择目标数据库连接
4. 点击"执行"

执行过程中会实时显示进度，包括已执行的语句数量和当前状态。

## 智能语句分割

SQL 文件中通常包含多条语句，DBX 使用智能分割逻辑来正确解析每条语句的边界，能够正确处理以下情况：

- **函数和存储过程** -- 包含 `BEGIN...END` 块的语句不会被错误地按分号拆分
- **触发器定义** -- 使用自定义分隔符的触发器语句能被正确识别
- **引号内的字符串** -- 字符串中包含的分号不会被当作语句分隔符

这确保了即使是结构复杂的 SQL 文件也能被正确执行。

## 取消执行

如果 SQL 文件较大或执行时间较长，你可以随时点击"取消"按钮中断执行。已执行的语句不会被回滚，仅停止后续语句的执行。
