# 表数据导入

DBX 支持将外部文件中的数据导入到数据库表中，适用于批量数据录入和数据迁移场景。

## 支持的文件格式

| 格式 | 说明 |
|---|---|
| CSV | 逗号分隔的文本文件 |
| TSV | 制表符分隔的文本文件 |
| JSON | JSON 数组格式 |
| Excel | `.xlsx` 格式的 Excel 文件 |

## 使用方式

1. 在目标表上右键，选择"导入数据"
2. 选择要导入的文件
3. 确认列映射关系
4. 选择导入模式
5. 点击"开始导入"

## 自动列映射

选择文件后，DBX 会自动将文件中的列与目标表的字段进行匹配：

- 根据列名（或表头名称）自动映射到同名字段
- 你可以手动调整映射关系，将文件中的列指向不同的目标字段
- 不需要导入的列可以取消映射

## 导入模式

DBX 提供两种导入模式：

- **追加模式** -- 将文件中的数据追加到表的现有数据之后，不影响已有记录
- **清空后导入** -- 先清空表中的所有数据，再导入文件内容。适用于需要完全替换表数据的场景
