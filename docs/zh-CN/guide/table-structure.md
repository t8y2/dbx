# 表结构编辑器

DBX 提供可视化的表结构编辑器，让你无需手写 DDL 语句即可管理表的列和索引。

## 列编辑

在表结构编辑器中，你可以通过可视化界面管理表的列：

- **添加列** -- 点击"添加列"按钮，设置列名、数据类型、是否允许 NULL、默认值等属性
- **修改列** -- 直接编辑已有列的属性，包括重命名、更改类型、调整默认值等
- **删除列** -- 选中列后点击删除，标记该列待移除

所有修改不会立即生效，而是暂存在本地等待确认。

## 索引管理

除了列之外，你还可以在同一界面中管理表的索引：

- 添加新索引，指定索引列和索引类型（普通索引、唯一索引等）
- 修改已有索引的配置
- 删除不再需要的索引

## SQL 预览

在点击"应用"之前，DBX 会展示即将执行的 `ALTER TABLE` 语句，让你确认每一项变更。你可以：

- 查看完整的 DDL 变更语句
- 确认无误后执行
- 如发现问题，返回继续调整

## 数据库支持

表结构编辑器支持以下数据库引擎，并针对各引擎的特性做了适配：

| 数据库 | 支持情况 |
|---|---|
| MySQL | 支持 |
| PostgreSQL | 支持 |
| SQLite | 支持 |
| SQL Server | 支持 |
